# 1.2. Influences

Which language influenced which other? The following graph shows the most important influence relationships between the 56 languages covered. An edge A → B means: "A significantly influenced B" — whether through syntax, semantics, type system, or central design decisions.

Click on a language to highlight its influences (<span style={{color: '#1a7a2e', fontWeight: 600}}>green</span> = incoming, <span style={{color: '#b71c1c', fontWeight: 600}}>red</span> = outgoing). Dashed lines indicate shared platforms.

:::info
The representation is limited to **essential, direct influences**. Many languages have additional, more indirect sources of inspiration that are not shown here. Where a language has no incoming edges, it is either an origin (like Fortran, Prolog) or its influences lie outside the 56 languages covered.
:::
