# 1.2. Einflüsse

Welche Sprache hat welche andere beeinflusst? Der folgende Graph zeigt die wichtigsten Einflussbeziehungen zwischen den 56 behandelten Sprachen. Eine Kante A → B bedeutet: „A hat B maßgeblich beeinflusst" — sei es durch Syntax, Semantik, Typsystem oder zentrale Designentscheidungen.

Klicke auf eine Sprache, um ihre Einflüsse hervorzuheben (<span style={{color: '#1a7a2e', fontWeight: 600}}>grün</span> = eingehend, <span style={{color: '#b71c1c', fontWeight: 600}}>rot</span> = ausgehend). Gestrichelte Linien zeigen geteilte Plattformen an.

:::info
Die Darstellung beschränkt sich auf die **wesentlichen, direkten Einflüsse**. Viele Sprachen haben zusätzliche, indirektere Inspirationsquellen, die hier nicht abgebildet sind. Wo eine Sprache keine eingehenden Kanten hat, ist sie entweder ein Ursprung (wie Fortran, Prolog) oder ihre Einflüsse liegen außerhalb der 56 behandelten Sprachen.
:::
